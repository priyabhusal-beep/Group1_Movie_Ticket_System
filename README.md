Group1_Movie_Ticket_System

Quick start (NetBeans + Ant)

1. Recommended location
   - Move the project out of OneDrive/Dropbox to a local folder, e.g.
     `C:\Projects\Group1_Movie_Ticket_System`.
2. Open in NetBeans
   - File → Open Project → select the folder above.
3. Clean & Build
   - Right-click project → Clean and Build.
4. Run
   - Right-click project → Run.

Notes
- Keep `src/`, `lib/` and `nbproject/` in source control. Exclude `build/`, `dist/`, and `.nbproject/private/`.
- If Clean & Build fails with "Unable to delete directory" on Windows: ensure no running JVMs, pause OneDrive sync, and rerun Clean & Build.
- For submissions zip the project folder (excluding `build/`/`dist/`).

Contact
- Include instructions for DB connection in a separate `config.example.properties` if required.
